MiniTV – Lettore IPTV per iOS

Importa le tue playlist M3U/M3U8, organizza i canali e guarda in modo fluido con un'interfaccia pulita e veloce.

✔️ Import facile Supporto a URL remoti e file locali. Riconosce M3U/M3U8 standard.
✔️ Riproduzione stabile Player nativo, passaggio rapido tra canali, ripresa automatica.
✔️ UI minimal Niente fronzoli: solo i tuoi canali, in evidenza.

Nota: l'app non include canali preinstallati. Usa solo sorgenti di cui possiedi i diritti di visione/redistribuzione.

Vedi un esempio di playlist M3U 
#EXTM3U
# NOTE: Playlist di esempio senza contenuti protetti. Sostituisci gli URL con sorgenti autorizzate.
#EXTINF:-1 tvg-id="" tvg-name="Canale Pubblico 1 (Esempio)" tvg-logo="" group-title="Italia", Canale Pubblico 1 (Esempio)
https://bitdash-a.akamaihd.net/content/sintel/hls/playlist.m3u8
#EXTINF:-1 tvg-id="" tvg-name="Canale Pubblico 2 (Esempio)" tvg-logo="" group-title="Italia", Canale Pubblico 2 (Esempio)
https://test-streams.mux.dev/x36xhzz/x36xhzz.m3u8
#EXTINF:-1 tvg-id="" tvg-name="Sostituisci con la tua sorgente autorizzata" tvg-logo="" group-title="Italia", Tuo Canale (Esempio)
https://example.com/tua-sorgente-autorizzata.m3u8
Esempio di playlist JSON
